About Us

Welcome to our Real Time Sign Language Translator

The project is solely for learning & training purposes

The Real-Time Sign Language Translator is a web application designed to bridge the communication gap between individuals using sign language and those unfamiliar with it. Using computer vision and machine learning, the application translates hand gestures from sign language into text in real time. This innovative tool aims to enhance inclusivity and accessibility in communication.

Feel free to explore our application and see how it can help in recognizing hand signs in real-time. If you have any questions or feedback, please don't hesitate to reach out to us.
